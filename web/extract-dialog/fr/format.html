Les formats d'extraits pris en charge sont les suivants :
  • Protocolbuffer (PBF) format
  • OSM XML, compressé avec gzip, bzip2 ou 7-zip (xz)
  • o5m, compressé avec gzip ou bzip2
  • format Garmin, avec des styles pour OSM, OpenFietsMap, OpenSeaMap, OpenTopoMap, et de loisirs (freizeitkarte). Le routage est activé. Note : quelques appareils anciens ne supportent pas le format Unicode (UTF-8) et certains récents ont un firmware DRM défectueux ("can not unlock map"). Dans ces cas particuliers, veuillez utiliser use le style ASCII.
  • ESRI Shapefile format
  • Osmand (OBF) format pour les appareils Android
  • mapsforge format pour les appareils Android
  • maps.me format pour les appareils Android et iOS
  • format libosmium OPL, GeoJSON, SQLite, text
  • SRTM Höhenlinien für Garmin, Osmand, et PBF
  • format BBBike Perl/Tk
