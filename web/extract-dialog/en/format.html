Supported output formats are: [more help]
  • Protocolbuffer Binary Format (PBF) format
  • OSM XML, compressed with gzip, bzip2 or 7-zip (xz)
  • o5m, compressed with gzip or bzip2
  • Garmin format, with styles for OSM, cycle map, OpenFietsMap, OpenSeaMap, OpenTopoMap, BBBike and leisure (freizeitkarte). Routing is enabled. Note: some older devices don't support Unicode (UTF-8) and some new devices have a broken DRM firmware ("can not unlock map"). In this special case please use the ASCII style.
  • Esri shapefile format
  • Scalable Vector Graphics (SVG) format
  • Osmand (OBF) format for Android devices
  • mapsforge format for Android and iOS devices
  • maps.me format for Android devices
  • libosmium OPL, GeoJSON, SQLite, text format
  • BBBike Perl/Tk bbd format
  • SRTM Contours for Garmin, Osmand, and PBF
