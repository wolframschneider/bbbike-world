BBBike Garmin custom map extracts

Install a custom Garmin map on your device

Installing a custom Garmin image on your device is easy:
  1. connect your Garmin with a USB cable to your computer (or your SD card). You will see the Garmin device as a disk/storage in the explorer
  2. download the zip file and open it. There is a gmapsupp.img file inside
  3. copy the gmapsupp.img file to your Garmin device in the folder /Garmin with the explorer (Windows) or finder (MacOS). You can do this also on the command line on Linux/FreeBSD.

That's all. You are done! See also Installing_the_map_onto_your_GPS

The BBBike extract service is platform independent - for Windows, MacOS, Linux, FreeBSD etc. There is no windows .exe installer, and there are no other special tools required.

View a custom Garmin map on your PC

You can open the custom Garmin image on your computer with QMapShack and view the map.

It should be possible with Garmin BaseCamp too if you connect your device.

Garmin internals

Routing is enabled for the Garmin maps. Depending on your device, it is possible to store more than one Garmin map (or with different styles) on your Garmin.

By default all generated Garmin maps from OpenStreetMap data will use the code page Unicode (UTF-8). However, some older devices don't support Unicode and some new devices have a broken DRM firmware ("can not unlock map"). In this special case please use the latin1 style. Garmin SRTM extracts use always the ASCII character set.

Garmin Styles

osmstandard OpenStreetMap style
cyclemap style similar to opencyclemap.org
leisuremap style from freizeitkarte-osm.de, for outdoor activities. The .img file is ~30% larger than the osm/cycle file
bbbikemap style from bbbike.de, for outdoor activities.
onroadA Garmin style for older devices with less space - usually 1/10 the size of Garmin OSM, see github.com/raintonr/mkgmap-bike-onroad, for outdoor activities / racing bikes.
ontrailA Garmin style for older devices with less space - usually 1/5 the size of Garmin OSM, see github.com/raintonr/mkgmap-bike-onroad, for outdoor activities / hike & bike.
openfietslitemap style from OpenFietsMap.nl, for outdoor activities. Lite version
openfietsfullmap style from OpenFietsMap.nl, for outdoor activities. Full version
opentopomap style from OpenTopoMap.org
oseammap style from OpenSeaMap.org, the free nautical chart
